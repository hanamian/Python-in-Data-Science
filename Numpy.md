## np.zeros(...)
``python
np.zeros(2)
```
Result:
```
array([0., 0.])
```
``python
np.zeros((3,2))
```
Result:
```
array([[0., 0.],
       [0., 0.],
       [0., 0.]])
```

## np.eye(...)
it results 2D array (matrix)
``python
np.eye(3)
```
Result:
```python
array([[1., 0., 0.],
       [0., 1., 0.],
       [0., 0., 1.]])
```
